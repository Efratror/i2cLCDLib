Controller-object Commands
Library Documentation » Sending messages to the touchscreen

Description

These functions can manipulate Controller-objects. These objects are defined in de Visual LCD-studio. It is possible to set the values of these controlles. In some cases it's also possible to send a request to get te current value of a Control-object.

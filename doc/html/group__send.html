Sending messages to the touchscreen
Library Documentation

Description

A send-command is a way of manipulating the touchscreen. This is done by sending a message over i2c. This message starts with a Hex-code of types BasicCmd or ObjectCmd. Followed by the nessisary parameters.

The commands are divided in to two groups. The first are the Basic Commands. These commands manipulate basic concepts of the touchscreen. The second type of commands are the Controller-object Commands. These are able to manipulate Controller-object of the touchscreen.

Reading messages from the touchscreen.

Description

The Hunda tech touchscreen is able to send messages to the controlling device. These messages are generated when a Button-, Checkbox-, Slider- or Edit-control are touched (The touch-function should be enabled). An other case when messages are send is when a "Get Controller Value" Command was send.

When the touchscreen want's to send a message the int-pin will be pulled low. Thus signaling a valid-command or a touch of a controll on the screen.
